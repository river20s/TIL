# NIC (Network Interface Controller, 또는 Network Interface Card)

컴퓨터를 네트워크에 연결하여 통신하기 위해 사용하는 하드웨어 장치. 네트워크 카드, 랜 카드, 물리 네트워크 인터페이스라고 하며, 네트워크 인터페이스 카드, 네트워크 어댑터, 네트워크 카드, 이더넷 카드 등으로도 부른다.<sup>[1](#footnote_1)</sup>
